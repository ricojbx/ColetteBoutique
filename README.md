# Colette Boutique

Welcome page for **Colette Boutique**, a curated gift shop in Boise, ID —
*one-of-a-kind finds for one-of-a-kind friends.*

- **Address:** 3122 S Bown Way, Boise, ID 83706
- **Instagram:** [@colette_theboutique](https://instagram.com/colette_theboutique)

## About this site

A single static page (plain HTML/CSS/JS, no build step). The hero features the
brand silhouette with the shop name in an elegant serif; a cross-fading strip
below rotates through the brand imagery.

- `index.html` — page markup
- `styles.css` — brand palette (sage green, blush pink, cream) and layout
- `script.js` — accent-strip cross-fade rotator
- `assets/` — brand images and favicon

## Run locally

Open `index.html` in a browser, or serve it:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```
